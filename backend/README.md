# Dissertation Registration API

Acesta este backend-ul RESTful pentru aplicația de înscriere la disertație. Aplicația permite studenților să trimită cereri și profesorilor să gestioneze sesiunile de înscriere.

## Tehnologii Utilizate
- **Runtime:** Node.js
- **Framework:** Express.js
- **Bază de date:** SQLite (Relațională)
- **ORM:** Sequelize

## Prerechizite
- Node.js instalat (v14+)
- NPM
